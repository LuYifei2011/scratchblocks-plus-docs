# Scratchblocks-Plus Documentation

Welcome to the official documentation for **Scratchblocks-Plus** – an enhanced fork of [scratchblocks](https://github.com/scratchblocks/scratchblocks) that extends the original library with powerful new features and improvements.

## What is Scratchblocks-Plus?

Scratchblocks-Plus allows you to render Scratch block code as images or interactive visualizations, making it perfect for:

- Creating tutorials and documentation
- Building educational content
- Showcasing code examples
- Supporting multiple languages

## Documentation Structure

- **For Developers**: Quick start guide, installation instructions, and integration examples for different environments (HTML, React, Node.js).
- **Syntax**: Extended syntax features specific to Scratchblocks-Plus, including matrix support and other enhancements.

## About This Documentation

The syntax section of this documentation covers features added by Scratchblocks-Plus only. For information about the original scratchblocks syntax, refer to the [Scratch Wiki](https://en.scratch-wiki.info/wiki/Block_Plugin/Syntax).
